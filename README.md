## Deployment

To deploy this project first clone the repo then run

- ```bash
  npm install
  ```
- ```bash
  ts-node .
  ```
or if you want to to re-run autoautomatically when you make changed, run
- ```bash
  nodemon
  ```
then on line 36, put your connection URI on the designated place.
